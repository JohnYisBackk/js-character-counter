# Character Counter ✨

## 🌐 Live Demo

👉 https://johnyisbackk.github.io/js-character-counter/

---

A modern character counter component built with pure HTML, CSS, and vanilla JavaScript.

This project features live progress tracking, dynamic UI states, auto-resizing textarea behavior, and clipboard copy support.

---

## 🚀 Features

* ✍️ Live character counting
* 📊 Dynamic progress bar
* 🟢 Success / 🟡 Warning / 🔴 Danger states
* 📋 Copy-to-clipboard support
* 📏 Auto-resizing textarea
* ❌ Clear text functionality
* ⚡ Real-time UI updates
* 🎨 Modern responsive design

---

## 🎮 How It Works

1. User types into the textarea
2. Character count updates instantly
3. Progress bar fills dynamically
4. UI state changes based on usage percentage
5. User can copy or clear text with one click
